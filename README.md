# Live Polling App using React.js Node.js, and Socket.io

Topics
------
* Node js
* Socket IO
* React
* Webpack
* es6
* babel
* react-router

## Requirements

* [Node.js](http://nodejs.org/)
* [npm](http://npmjs.com)

## Installation

1. Clone the repo
2. Install dependencies: `npm install`
3. Start the server: `npm start`
4. Navigate to: `http://localhost:3000`
